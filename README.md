This is a simple HTMX element that can be used to toggle the dark mode on a site.

## Usage

```html
<htmx-dark-mode></htmx-dark-mode>
```

## Attributes

- `dark-theme` - The name of the dark theme (default: dark)
- `light-theme` - The name of the light theme (default: light)

## Installation

```bash
npm install @htmx/htmx-dark-mode
```

Use as a module in your application:

```html
<script src="https://unpkg.com/@htmx/htmx-dark-mode@latest/dist/index.js" type="module"></script>
```
